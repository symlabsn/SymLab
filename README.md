# SymLab

Une plateforme E-learning pour l'expérimentation mathématiques, ingénierie et physique pour la promotion des STEM au Sénégal.

## Symlabsn  MVP

Plateforme STEM interactive : Maths, Physique, Python, QCM, TPs.

## Lancer en local

```
npm install
npm run dev
```

## Déployer sur Vercel

- Importer ce repo GitHub
- Vercel détecte Next.js automatiquement
