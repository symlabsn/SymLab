Jour 071 — Cinématique 2R

Thème détaillé — code, sortie attendue et exercices.

Description

Exemple pédagogique ciblé sur Cinématique 2R. Le code suivant est prêt à être collé dans un notebook Python avec SymPy.

Code SymPy

from sympy import symbols, Matrix, cos, sin
th1,th2,l1,l2 = symbols('th1 th2 l1 l2')
x = l1*cos(th1)+l2*cos(th1+th2)
y = l1*sin(th1)+l2*sin(th1+th2)
Matrix([x,y])
Sortie attendue : \(\begin{pmatrix}x\\y\end{pmatrix}\)
