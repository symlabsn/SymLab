Jour 005 — Affichage mathématique

Thème détaillé — code, sortie attendue et exercices.

Description

Exemple pédagogique ciblé sur Affichage mathématique. Le code suivant est prêt à être collé dans un notebook Python avec SymPy.

Code SymPy

from sympy import symbols, integrate, sin, pi
x = symbols('x')
integrate(sin(x)/x, (x,0,pi))
Sortie attendue : \(\int_{0}^{\pi}\frac{\sin x}{x}\,dx\)
